# 👋 Hi, I'm Daniel

I work at the intersection of **ERP systems, data, and business decisions**.

Over the past 3+ years, I’ve been responsible for turning complex, messy ERP data (Dynamics 365 F&O) into structured, reliable insights used by leadership across multiple regions.

---

## 🧠 How I think about data

I don’t see data as dashboards — I see it as **operational leverage**.

* If reporting takes too long → automate it
* If data is inconsistent → govern it
* If teams don’t trust it → fix the source, not the visualization

Most of my work has been focused on making data **usable, reliable, and fast enough to matter**.
